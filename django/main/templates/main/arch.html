ARCHITECTURE

The system pipeline is built from 4 concatenated modules.
An input image is sent by the user and then it passes through:

  • Face detection module
  • VGG Face descripotr module
  • Grad-CAM module
  • Feature extraction module

The output is then sent back and presented to the user.

A more thorough description is provided in the design document.

VGG-Face Descriptor - a Convolutional Neural network (CNN) bsaed on the VGG-Very-Deep-16 network which is at a state-of-the-art level at face recognition.

Reference: http://www.robots.ox.ac.uk/~vgg/software/vgg_face/

Gradient-weighted Class Activation Map (Grad-CAM) - A technique for producing "visual explenation" for decisions from a large class of CNN-based models, making them more transperent. Grad-CAM uses the gradients of any target concept, flowing into the final convolutional layer to produce a coarse localization map highlighting the important regions in the image for predicting the concept. Unlike previous approaches. Grad-CAM is applicable for a wide variety of CNN model-families.

Reference: https://arxiv.org/abs/1610.02391
